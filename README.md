学生资金管理助手 - Coze 智能体
项目简介
面向大学生群体的轻量化资金管理智能体，基于 Coze 平台零代码实现，无需自定义函数，核心解决学生记账繁琐、投资知识匮乏、预算失控等痛点。

核心功能
趣味互动交流：通过 Coze 内置大模型，用生活化语言解答财务疑问，缓解资金焦虑；
账目管理：上传账单图片，自动归类消费类型，生成月度预算；
投资教程学习：基于知识库推荐低风险投资入门内容，简化专业术语；
低风险投资建议：结合合规财经数据插件，推荐学生适配的低门槛理财产品。
技术实现（无自定义函数）
大模型：Coze 内置大模型（或对接 ERNIE 3.0 Tiny 轻量化模型）；
核心工具：PP-OCRv4 插件（账单识别）、财经数据插件（投资产品数据）、Coze 内置工作流（步骤编排）、知识库（教程存储）；
核心逻辑：通过 Coze 工作流串联“用户输入→插件调用→大模型处理→结果返回”，无需编写自定义函数。
